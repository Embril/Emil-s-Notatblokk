HTML Attributes


HTML attributes provide additional information about HTML elements.


All HTML elements can have attributes. Attributes provide additional information about elements. Attributes are always specified in the start tag. attributes usually come in name/value pairs like: name="value"


The href attribute


The a tag defines a hyperlink. the href attribute specifies the URL of the page the link goes to: a href="https://www.w3schools.com"


The scr attribute


The img tag is used to embed an image in an HTML page. the scr attribute specifies the path to the image to be displayed. EKS: img scr="img_girl.jpg"


The img tag shuld also contain the width and height attributes, which specifies the width and height of the image (in pixels)


The required alt attribute for some img tag specifies an alternate text for an image, if the image for some reason cannot be displayed. This can due to slow connection, or an error in the scr attribute, or if the user uses a screen reader.


The style attribute


The style attribute is used to add styles to an element, such as clor, font, size and more.


The lang attribute


You should always include the lang attribute inside the html tag, to declare the language of the web page. this is meant to assist search engines and browsers The following example specifies English as the language: html lang="en"


Country codes can also be added to the language code in the lang attribute. si, the first two characters define the language of the HTML page, and the last two characters define the country. The following example specifies English language and the United States as the country: html lang="en-US"
